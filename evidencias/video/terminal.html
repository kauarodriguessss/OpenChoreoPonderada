openchoreo-quick-start — instalação OpenChoreo v1.1.1 (macOS · Docker Desktop)
# 1) Imagem quick-start e container
$ docker pull ghcr.io/openchoreo/quick-start:v1.1.1
Status: Downloaded newer image for ghcr.io/openchoreo/quick-start:v1.1.1
$ docker run -d -it --network=host -v /var/run/docker.sock:/var/run/docker.sock ...
# 2) Instalação (como usuário openchoreo)
$ docker exec -u openchoreo openchoreo-quick-start ./install.sh --version v1.1.1
[INFO] k3d cluster 'openchoreo-quick-start' created successfully
[INFO] Helm: cert-manager · external-secrets · kgateway · openbao · thunder
[INFO] Helm: openchoreo-control-plane · openchoreo-data-plane (v1.1.1)
[SUCCESS] OpenChoreo installation completed successfully!
# 3) Validação
$ docker exec -u openchoreo openchoreo-quick-start ./validate-installation.sh
[SUCCESS] All validations passed!
23 pods Running · Control Plane [READY] · Data Plane [READY]
# 4) Deploy da aplicação de exemplo
$ docker exec -u openchoreo openchoreo-quick-start ./deploy-react-starter.sh
